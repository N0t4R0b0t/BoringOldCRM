# Assistant-Driven Tenant Setup Prompts

Three prompts for full demo-tenant setup. Each prompt uses **bulk tool calls** for all sample data — JSON arrays are passed directly so the AI skips interpretation, minimising token usage.

**Budget per prompt**: ~5 bulk calls for data (customers / contacts / opportunities / activities / custom-records), plus a single `bulkCreateCustomFields` call. No per-record loops.

> **Tech note**: Calculated field expressions are **CEL**. Policy rules are **Rego** (OPA). The AI knows both — just describe the logic and it generates the correct syntax.

Simply paste each prompt into `/chat` or `/onboarding` in **auto mode**.

---
